# Security Policy

## Supported Versions

The following versions of the UKSFTA DevOps Suite are currently supported with security updates:

| Version | Supported          |
| ------- | ------------------ |
| v1.x    | :white_check_mark: |
| < v1.0  | :x:                |

## Reporting a Vulnerability

We take the security of our project seriously. If you believe you have found a security vulnerability, please report it by opening a private security advisory on GitHub or by contacting the project leadership directly through established internal channels.

Do not open a public issue for security-related reports.
